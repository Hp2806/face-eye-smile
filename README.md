## Face, Eye & Smile Detection using OpenCV

A real-time multi-feature detection project built with Python and OpenCV. The program detects faces, eyes, and smiles simultaneously through the webcam and draws color-coded labeled bounding boxes around each detected feature in real time using Haar Cascade Classifiers.

## Features

- Real-time face, eye, and smile detection through webcam
- Multi-feature detection using Haar Cascade Classifiers
- Histogram equalization for better detection in low-light conditions
- Region-of-interest cropping to minimize false positives
- Color-coded labeled bounding boxes for each detected feature
- Automatic periodic snapshot saving when a face is present
- Smile burst capture mode


## Technologies Used

- Python
- OpenCV
- NumPy
